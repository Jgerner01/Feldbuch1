# Feldbuch1
Feldbuch-Software mit AutoCAD LT
- Ziel ist eine Verbindung zu einem Leica-Tachymeter (TCR307) herzustellen, und ein GUI Modul mit dem man Punktnummer und Code festlegen kann und in dem man unterschiedliche Prismenkonstanten auswählen kann. Ggf soll damit auch der Pilotlaser aktiviert werden können. 
- eine Frei Stationierung zu nutzen, welche die Koordinaten der Identischen Punkte über AutoCAD LT empfängt und die Messdaten vom Tachymeter in Echtzeit einließt.
- gut wäre auch, dass der aktuelle Winkel im AutoCAD LT dargestellt wird.
- Speicherung aller Messdaten in eine ASCII Datei mit der Endung .dat
- Speicherung der Koordinaten in eine ASCII Datei mit der Endung .kor
